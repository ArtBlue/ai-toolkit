## Methodology

This skill synthesizes findings across:

- **Academic research:** Transformer attention, prompt engineering, code generation
- **Framework documentation:** Version-specific breaking changes
- **Information architecture:** Signal-to-noise, foraging theory

## How This Research Informs the Auditor

| Research Area | Audit Criteria |
|---------------|----------------|
| Attention distribution (Liu et al.) | Primacy zone optimization (first 2K tokens) |
| Prompt caching (Anthropic) | Static content, <2K threshold, cache efficiency scoring |
| Code generation fidelity (Microsoft) | Correctness Guards for version-specific syntax |
| Context utilization (UC Berkeley) | Discovery-first architecture, signal-to-noise |
| WET principle (Khachatryan) | Fidelity > DRY scoring |
| Instruction following (DeepMind) | Attention architecture layers |
